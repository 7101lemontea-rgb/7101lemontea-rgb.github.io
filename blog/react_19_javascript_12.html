JavaScript 12 — 구조 분해 할당

배열이나 객체의 속성을 해체하여 개별 변수에 효율적으로 담는 구조 분해 할당(Destructuring)을 알아봅니다.

목차

  • 1.개요 및 Why React?
  • 2.객체 구조 분해
  • 3.배열 구조 분해
  • 4.함수 인자 및 React 활용
  • ※연습 문제

01

구조 분해 할당 (Destructuring Assignment)

배열이나 객체의 속성을 해체하여 그 값을 개별 변수에 담을 수 있게 하는 표현식입니다.

Why React?

  • props 활용: 부모가 전달하는 props 객체를 받아올 때 props.name 대신 구조 분해를 사용하여 코드를 간결하게 만듭니다.
  • Hooks 사용: useState를 사용할 때 반환되는 배열을 처리하기 위해 필수적으로 사용됩니다.

02

객체 구조 분해

객체의 키(key) 이름을 기준으로 변수에 할당하며, 순서는 상관없습니다. 

const user = { name: "김철수", age: 30, email: "kim@example.com" };

// 1. 기본 구조 분해 할당
const { name, email } = user;

// 2. 변수명 변경 (key: 새변수명)
const { name: userName, email: userEmail } = user;

// 3. 기본값 설정
const { name, phone = "없음" } = user;

03

배열 구조 분해

배열의 인덱스 순서를 기준으로 변수에 할당합니다. 

const colors = ["red", "green", "blue"];

// 인덱스 순서대로 할당
const [first, second, third] = colors;

// 일부 값 무시 (쉼표 사용)
const [firstColor, , thirdColor] = colors; // green 건너뜀

04

함수 및 React에서의 활용

함수 인자 구조 분해

// 인자에서 바로 객체 해체
function printUser({ name, age }) {
  console.log(name, age);
}

React 코드 예시

// 컴포넌트 Props 받기
const UserCard = ({ name, age, email }) => {
  return <div>{name}</div>;
};

// Hook 사용 시 배열 구조 분해
const [count, setCount] = useState(0);

연습 문제

Q1. 객체 구조 분해

const person = { name: "홍길동", age: 30, city: "서울" };
const { name, city } = person;

Q2. 화살표 함수와 템플릿 리터럴

const printUser = ({ name, age }) => {
  console.log(`이름 : ${name}, 나이 : ${age}`);
};

Q3. 배열 인자 합계 계산

const sumFirstTwo = ([first, second]) => {
  return first + second;
};

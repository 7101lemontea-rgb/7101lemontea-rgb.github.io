Overview

CSS란?

CSS(Cascading Style Sheets)는 웹 페이지의 디자인과 레이아웃과 같은 스타일 규칙들을 체계적으로 관리하고 적용하는 '스타일링' 언어입니다. 텍스트의 크기나 색상, 이미지의 크기, 웹페이지의 배경색, 요소들의 배치 등을 결정하여 웹페이지를 시각적으로 아름답고 사용하기 편리하게 만들어줍니다.

🎨
디자인

색상, 폰트, 크기 등 시각적 요소를 제어합니다.

📐
레이아웃

요소들의 위치와 배치를 결정합니다.

Section 01

CSS의 속성

CSS는 기본적으로 '무엇을(선택자)', '어떤 부분을(속성)', '어떻게(값)' 바꿀 것인지 지정하는 방식으로 작성됩니다.

CSS
선택자 {
  속성: 값;
}
선택자 Selector

디자인을 적용할 HTML 대상을 가리킵니다. 예) h1, p 등

속성 Property

바꾸고 싶은 스타일의 종류입니다. 예) 글자색 color, 글자크기 font-size

값 Value

속성에 적용할 구체적인 수치나 상태입니다. 예) 파란색 blue, 24px

Section 02

CSS 작성 방법

CSS를 HTML에 적용하는 방법은 3가지가 있습니다.

2.1 인라인 스타일

태그에 직접 작성합니다. 특정 요소에만 스타일을 적용할 때 사용합니다.

HTML
<p style="color: red; font-size: 20px;">빨간 글자</p>
✅ 장점

빠르게 테스트 가능

❌ 단점

재사용 불가, 유지보수 어려움

2.2 내부 스타일 시트

<head> 안에 <style> 태그를 사용하여 작성합니다.

HTML
<!DOCTYPE html>
<html>
  <head>
    <style>
      p {
        color: blue;
        font-size: 18px;
      }
    </style>
  </head>
  <body>
    <p>이 문단은 파란색입니다.</p>
  </body>
</html>
  • 해당 HTML 문서에만 적용됩니다.
  • 다른 페이지에서 재사용 불가합니다.

2.3 외부 스타일 시트

별도의 .css 파일로 분리합니다.

HTML
<!DOCTYPE html>
<html>
  <head>
    <link rel="stylesheet" href="style.css">
  </head>
  <body>
    <p>외부 CSS 파일 사용</p>
  </body>
</html>
CSS — style.css 
/* style.css */
p {
  color: green;
  font-size: 16px;
}
✅ 장점

여러 페이지에서 재사용 가능

유지보수가 쉽습니다.

❌ 단점

파일 관리가 필요합니다.

실제 프로젝트에서는 외부 스타일 시트를 가장 많이 사용합니다. 유지보수와 재사용성이 가장 뛰어나기 때문입니다.

Section 03

선택자 (Selector)

선택자는 '어떤 요소에 스타일을 적용할 것인가'를 지정합니다.

3.1 태그 선택자

HTML 태그 이름을 그대로 선택자로 사용합니다. 해당 태그 전체에 스타일이 적용됩니다.

CSS
/* 모든 p 태그에 적용 */
p {
  color: blue;
}

/* 모든 h1 태그에 적용 */
h1 {
  font-size: 32px;
}
HTML
<p>이 문단은 파란색입니다.</p>
<p>이 문단도 파란색입니다.</p>
<h1>이 제목은 32px입니다.</h1>

3.2 class 선택자

  • 여러 요소에 동일한 스타일을 적용할 때 사용합니다.
  • 점(.)으로 시작합니다.
CSS
/* class 이름 앞에 점(.) 붙이기 */
.highlight {
  background-color: yellow;
  font-weight: bold;
}

.big-text {
  font-size: 24px;
}
HTML
<p class="highlight">강조된 문단</p>
<div class="highlight">강조된 div</div>
<span class="big-text">큰 글씨</span>

<!-- 여러 class 동시 적용 가능 -->
<p class="highlight big-text">강조되고 큰 문단</p>

하나의 요소에 여러 class를 동시에 적용할 수 있습니다. 공백으로 구분하여 작성합니다.

여러 class를 조합해 버튼처럼 공통 스타일과 변형 스타일을 분리할 수 있습니다.

CSS
.btn {
  padding: 10px 20px;
  border: none;
  border-radius: 5px;
  cursor: pointer;
}

.btn-primary {
  background-color: #007bff;
  color: white;
}

.btn-danger {
  background-color: #dc3545;
  color: white;
}
HTML
<button class="btn btn-primary">저장</button>
<button class="btn btn-danger">삭제</button>

3.3 id 선택자

  • 페이지에서 단 하나의 요소에만 사용합니다.
  • 샵(#)으로 시작합니다.
CSS
/* id 이름 앞에 샵(#) 붙이기 */
#header {
  background-color: navy;
  color: white;
  padding: 20px;
}

#footer {
  background-color: #333;
  color: white;
  text-align: center;
}
HTML
<!-- id는 페이지에서 한 번만 사용 -->
<div id="header">
  <h1>웹사이트 헤더</h1>
</div>

<div id="footer">
  <p>© 2024 My Website</p>
</div>
선택자 기호 적용 대상 예시
태그 선택자 — 해당 태그 전체 p { }
class 선택자 . 같은 class를 가진 여러 요소 .highlight { }
id 선택자 # 페이지 내 단 하나의 요소 #header { }
Section 04

색상

CSS에서 색상을 지정하는 방법은 여러 가지가 있습니다.

4.1 색상 이름

영문 색상 이름을 그대로 사용합니다. 직관적이지만 표현 가능한 색상이 제한적입니다.

CSS
/* 1. 색상 이름 */
color: red;
color: blue;
color: tomato;

4.2 HEX 코드 (16진수)

가장 많이 사용하는 방식입니다. #RRGGBB 형식으로 빨강·초록·파랑을 각각 00~ff로 지정합니다.

CSS
/* 2. HEX 코드 (16진수) - 가장 많이 사용 */
color: #ff0000;  /* 빨강 */
color: #00ff00;  /* 초록 */
color: #0000ff;  /* 파랑 */
color: #333333;  /* 진한 회색 (줄여서 #333) */

4.3 RGB

빨강·초록·파랑을 각각 0~255 숫자로 지정합니다.

CSS
/* 3. RGB */
color: rgb(255, 0, 0);      /* 빨강 */
color: rgb(100, 100, 100);  /* 회색 */

4.4 RGBA — 투명도 포함

RGB에 투명도(alpha) 값을 추가합니다. 0(완전 투명) ~ 1(완전 불투명) 사이의 값을 사용합니다.

CSS
/* 4. RGBA - 투명도 포함 (0~1) */
/* 배경 이미지가 있거나 다른 요소 위에 겹칠 때 → 반투명 유리창 느낌 */
color: rgba(255, 0, 0, 0.5);  /* 반투명 빨강 */
방식 예시 특징
색상 이름 red, blue, tomato 직관적이나 색상 종류 제한
HEX #ff0000 가장 많이 사용, 정밀한 색 지정
RGB rgb(255, 0, 0) 0~255 숫자로 직관적 조합
RGBA rgba(255, 0, 0, 0.5) 투명도 지정 가능
Section 05

텍스트 스타일링

5.1 글꼴 스타일

CSS
/* 폰트 크기 */
font-size: 16px;    /* px, em, rem, % 등 */

/* 폰트 굵기 */
font-weight: bold;  /* normal, bold, 100~900 */

/* 폰트 스타일 */
font-style: italic; /* normal, italic */
속성 주요 값 설명
font-size 16px, 1rem, 1em, 100% 글자 크기
font-weight normal, bold, 100~900 글자 굵기
font-style normal, italic 글자 기울기

5.2 텍스트 레이아웃 스타일

CSS
/* 텍스트 정렬 */
text-align: center;      /* left, center, right, justify */

/* 텍스트 장식 */
text-decoration: underline; /* none, underline, line-through */

/* 줄 간격 */
line-height: 1.6;         /* 1.5~2.0 권장 */
속성 주요 값 설명
text-align left, center, right, justify 텍스트 가로 정렬
text-decoration none, underline, line-through 텍스트 꾸미기 (밑줄, 취소선 등)
line-height 1.5, 1.6, 2.0 줄 간격 (1.5~2.0 권장)

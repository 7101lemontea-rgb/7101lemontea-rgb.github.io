함수 (Function)

함수 개요

함수는 특정 작업을 수행하는 코드 블록입니다. JavaScript에서 함수를 작성하는 방법은 크게 세 가지로 나뉩니다.

선언문
함수 선언문 (Function Declaration)

function 키워드로 정의. 호이스팅(Hoisting)이 적용되어 선언 전에 호출해도 동작합니다.

표현식
함수 표현식 (Function Expression)

함수를 변수에 값으로 할당하는 방식. 표현식은 값으로 평가될 수 있는 것으로, 변수에 할당할 수 있습니다.

화살표
화살표 함수 (Arrow Function)

ES6에서 도입된 간결한 함수 문법. =>를 사용하며 단일 표현식은 return과 중괄호를 생략할 수 있습니다.

01

함수 선언문

a. 함수 선언

js
function greet(name) {
  console.log(`안녕하세요, ${name}님!`);
}

greet("홍길동"); // 안녕하세요, 홍길동님!
greet("김철수"); // 안녕하세요, 김철수님!

b. 반환값이 있는 함수

js
function add(a, b) {
  return a + b;
}

let result = add(5, 3);
console.log(result); // 8

c. 기본 매개변수

매개변수에 기본값을 설정하면 인수를 전달하지 않았을 때 기본값이 사용됩니다.

js
function introduce(name, age = 20) {
  console.log(`이름: ${name}, 나이: ${age}세`);
}

introduce("홍길동");      // 이름: 홍길동, 나이: 20세  ← age 기본값 사용
introduce("김철수", 25); // 이름: 김철수, 나이: 25세  ← 전달한 값 사용

실습 예제

예제 1 — 짝수 판별

js
function isEven(num) {
  return num % 2 === 0;
}

console.log(isEven(4)); // true
console.log(isEven(1)); // false

예제 2 — 할인가 계산 함수

js
function calculateDiscount(price, discountRate) {
  let discountAmount = price * (discountRate / 100);
  let finalPrice = price - discountAmount;
  return finalPrice;
}

예제 3 — 평균 계산 함수

js
function calculateAverage(scores) {
  let sum = 0;
  for (let i = 0; i < scores.length; i++) {
    sum += scores[i];
  }
  return sum / scores.length;
}

예제 4 — 최댓값 찾기 함수

js
function findMax(numbers) {
  let max = numbers[0];
  for (let i = 1; i < numbers.length; i++) {
    if (numbers[i] > max) {
      max = numbers[i];
    }
  }
  return max;
}

02

함수 표현식

표현식은 값으로 평가될 수 있는 것으로, 변수에 할당할 수 있습니다. 함수를 변수에 값처럼 저장하는 방식입니다.

js
const add = function (x, y) {
  return x + y;
};

console.log(add(1, 2)); // 3
함수 선언문

호이스팅 적용 — 선언 전에 호출 가능합니다.

greet(); // ✅ 정상 동작
function greet() { ... }
함수 표현식

호이스팅 미적용 — 선언 후에만 호출 가능합니다.

greet(); // ❌ 오류 발생
const greet = function() { ... };

03

화살표 함수

화살표 함수는 ES6에서 도입된 간결한 함수 문법입니다. =>를 사용하여 작성하며, 단일 표현식일 때는 return과 중괄호를 생략할 수 있습니다.

방식 코드 설명
기존 방식 function add(a, b) { return a + b; } function 키워드로 선언
화살표 함수 const add = (a, b) => { return a + b; }; => 사용, 함수 표현식 형태
중괄호 생략 const add = (a, b) => a + b; 한 줄이면 return과 중괄호 생략 가능
괄호 생략 const double = x => x * 2; 매개변수가 하나면 괄호도 생략 가능
js
// 기존 방식
function add(a, b) {
  return a + b;
}

// 화살표 함수
const add = (a, b) => {
  return a + b;
};

// 더 간결하게 (한 줄이면 중괄호와 return 생략 가능)
const add = (a, b) => a + b;

// 매개변수가 하나면 괄호도 생략 가능
const double = x => x * 2;

실습 예제

예제 1 — 성인 여부 확인

js
const isAdult = (age) => age >= 18;

console.log(isAdult(20)); // true
console.log(isAdult(15)); // false

예제 2 — 배열에서 짝수만 찾기

js
const numbers = [1, 2, 3, 4, 5, 6];
const isEven = (num) => num % 2 === 0;

for (const num of numbers) {
  if (isEven(num)) {
    console.log(num); // 2, 4, 6
  }
}

연습 문제 1

원의 넓이 구하는 화살표 함수

반지름을 입력받아 원의 넓이를 구하는 화살표 함수를 작성하세요.

js
// 방법 1: 3.14 사용
const area = (r) => 3.14 * r * r;

// 방법 2: Math.PI 사용 (더 정확)
const area2 = (r) => Math.PI * r * r;

console.log(area(5));  // 78.5
console.log(area2(5)); // 78.53981633974483

연습 문제 2

함수를 화살표 함수로 변환하기

다음 함수들을 화살표 함수로 변환하세요.

(1) 제곱 계산

변환 전 — 함수 선언문
function square(n) {
  return n * n;
}
console.log(square(5)); // 25
변환 후 — 화살표 함수
const square = n => n * n;

console.log(square(5)); // 25

(2) 인사말 생성 (기본 매개변수 포함)

변환 전 — 함수 선언문
function greet(name, greeting = "안녕하세요") {
  return `${greeting}, ${name}님!`;
}
console.log(greet("홍길동"));
// 안녕하세요, 홍길동님!
console.log(greet("김철수", "반갑습니다"));
// 반갑습니다, 김철수님!
변환 후 — 화살표 함수
const greet = (name, greeting = "안녕하세요") =>
  `${greeting}, ${name}님!`;

console.log(greet("홍길동"));
// 안녕하세요, 홍길동님!
console.log(greet("김철수", "반갑습니다"));
// 반갑습니다, 김철수님!

(3) 합격 여부 판별

변환 전 — 함수 선언문
function isPassed(score) {
  return score >= 60;
}
console.log(isPassed(75)); // true
변환 후 — 화살표 함수
const isPassed = score => score >= 60;

console.log(isPassed(75)); // true

매개변수가 하나이고 반환식도 한 줄이면 괄호와 return, 중괄호를 모두 생략하여 const 함수명 = 매개변수 => 표현식; 형태로 가장 간결하게 쓸 수 있습니다.

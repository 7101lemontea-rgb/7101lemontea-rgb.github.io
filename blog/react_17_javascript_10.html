01

단축 평가 (Short-circuit Evaluation)

논리 연산자 &&, ||가 좌측 피연산자의 값만 보고 전체 결과를 확정하여, 필요한 경우에만 우측 피연산자를 평가하는 방식입니다.

1.1. A && B (AND)

왼쪽(A)이 Truthy일 때만 오른쪽(B)을 반환합니다. 

const isAdmin = true;
isAdmin && console.log("관리자 메뉴 접근 가능");
// isAdmin이 true일 때만 메시지가 출력됩니다.

1.2. A || B (OR)

왼쪽(A)이 Falsy일 때 오른쪽(B)을 반환합니다. 주로 기본값을 설정할 때 사용합니다. 

const isLoggedIn = false;
isLoggedIn || console.log("로그인이 필요합니다.");
// 로그인하지 않은 경우(false)에만 메시지가 출력됩니다.

※ 실습: 단축 평가 활용하기

// 1. 존재 여부 확인 후 출력
user && user.name && console.log(`안녕하세요, ${user.name}님!`);

// 2. 기본값 출력
const displayName = nameA || "이름 없음";

02

배열 메서드 (map & filter)

데이터 목록을 다루는 리액트 환경에서 가장 중요하게 다뤄지는 메서드들입니다.

2.1. map()

배열의 모든 요소를 순회하며 콜백 함수의 실행 결과를 모아 새로운 배열을 반환합니다.

[Why React?] 데이터 목록을 JSX 엘리먼트로 변환하여 화면에 렌더링할 때 필수적으로 사용됩니다.

js
const numbers = [1, 2, 3];
const doubled = numbers.map(num => num * 2); // [2, 4, 6]

// 객체 배열에서 특정 속성만 추출
const names = users.map(user => user.name);

2.2. filter()

배열 요소 중 콜백 함수의 조건이 true인 요소만 골라 새로운 배열을 반환합니다.

js
const numbers = [1, 2, 3, 4, 5];
const evens = numbers.filter(num => num % 2 === 0); // [2, 4]

2.3. 메서드 체이닝 (Chaining)

filter와 map을 연결하여 특정 조건의 데이터를 추출하고 가공하는 과정을 한 번에 처리할 수 있습니다. 

const availableNames = products
  .filter(p => p.inStock)
  .map(p => p.name);

연습 문제 종합

// 1. 80점 이상 필터링
const highScore = scores.filter(s => s >= 80);

// 2. 2학년 이상 학생 이름 추출
const results = students
  .filter(s => s.grade >= 2)
  .map(s => s.name);

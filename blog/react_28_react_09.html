React 09 — 제어 컴포넌트

폼 입력값을 React의 State로 관리하여 데이터의 무결성을 유지하는 방법을 학습합니다.

목차

  • 1.제어 컴포넌트의 이해
  • 2.e.target 속성
  • 3.입력값 State 관리
  • 4.실습: 입력 폼 제어
  • 5.여러 개의 Input 관리
  • 6.실습: 사용자 프로필 카드

01

제어 컴포넌트 (Controlled Components)

제어 컴포넌트는 폼 입력값을 React state로 관리하는 컴포넌트입니다. 일반 HTML에서는 입력값을 DOM이 직접 관리하지만, React에서는 state가 입력값을 관리하도록 만들 수 있습니다.

핵심 구조:

User Input ➔ onChange(e) ➔ setState(e.target.value) ➔ State Update ➔ Re-render ➔ Input Value Update

02

e.target

e.target은 이벤트가 실제로 발생한 DOM 요소 자체를 가리키며, 이 요소에 달려 있는 HTML 속성들에 바로 접근할 수 있습니다. 

onChange={(e) => {
  console.log(e.target);
  // → <input name="name" id="name" /> (이 input 요소 자체)
}}

※ e.target에서 자주 꺼내 쓰는 속성

속성 설명 예시 값
e.target.value 사용자가 입력한 값 "홍길동"
e.target.name input의 name 속성 "name"
e.target.id input의 id 속성 "name"
e.target.type input의 type 속성 "text", "checkbox"
e.target.checked 체크박스의 체크 여부 true / false

03

입력값 State 관리

입력창의 value 속성에 state 값을 연결해야 React가 해당 요소를 완전히 제어할 수 있습니다. 

import React, { useState } from "react";

function InputControl() {
  const [name, setName] = useState("");

  return (
    <div>
      <label htmlFor="name">이름:</label>
      <input
        name="name"
        id="name"
        value={name}
        onChange={(e) => setName(e.target.value)}
      />
    </div>
  );
}

04

실습: 입력 폼 제어

텍스트 미러링, 대문자 변환기, 글자수 출력, 그리고 초기화 기능을 포함한 예제입니다. 

function ExampleInput() {
  const [userId, setUserId] = useState("");

  return (
    <div className="m-16 g-8">
      <div>
        <label htmlFor="userId" className="m-4 text-2xl">ID</label>
        <input
          type="text"
          value={userId}
          onChange={(e) => setUserId(e.target.value.toUpperCase())}
        />
        <button onClick={() => setUserId("")}>초기화</button>
      </div>
      <h3>{userId}</h3>
      <h3>글자 수 : {userId.length}</h3>
    </div>
  );
}

유효성 검사 실습

비밀번호가 8자리 이상이며 영문, 숫자, 특수문자를 포함하는지 실시간으로 검사합니다. 

const validatePassword = (password) => {
  return (
    password.length >= 8 &&
    /[a-zA-Z]/.test(password) &&
    /[0-9]/.test(password) &&
    /[!@#$%^&*()]/.test(password)
  );
};

05

여러 개의 Input 다루기

input이 많아지면 개별 useState 대신 객체로 관리하는 것이 깔끔합니다. 

const [form, setForm] = useState({ name: "", email: "" });

const handleChange = (e) => {
  const { name, value } = e.target;
  setForm({
    ...form, // 기존 값 복사 (Spread Operator)
    [name]: value // computed property name 사용
  });
};

06

실습: 사용자 프로필 카드

이름, 전화번호, 이메일을 입력받아 실시간으로 프로필 카드를 업데이트합니다. 

import React, { useState } from "react";

function ProfileCard() {
  // 1. 객체 형태의 state 초기화
  const [userInfo, setUserInfo] = useState({
    username: "",
    phone: "",
    email: "",
  });

  // 2. 통합 이벤트 핸들러
  const handleChange = (e) => {
    const { name, value } = e.target;
    // 기존 정보를 복사(...userInfo)하고 변경된 name 키값만 업데이트
    setUserInfo({ ...userInfo, [name]: value });
  };

  // 3. 구조 분해 할당으로 값 추출
  const { username, phone, email } = userInfo;

  return (
    <div className="max-w-md mx-auto mt-10 p-6">
      <h3 className="mb-5 text-gray-800 text-2xl font-semibold">
        사용자 프로필
      </h3>
      
      <div className="mb-4">
        <label className="block mb-1.5 font-semibold text-gray-600">이름</label>
        <input
          name="username"
          value={username}
          onChange={handleChange}
          className="w-full border p-2.5 rounded focus:outline-none"
        />
      </div>

      <div className="mb-4">
        <label className="block mb-1.5 font-semibold text-gray-600">전화번호</label>
        <input
          name="phone"
          type="tel"
          value={phone}
          onChange={handleChange}
          className="w-full border p-2.5 rounded focus:outline-none"
        />
      </div>

      {/* 실시간 프로필 카드 출력 */}
      <div className="mt-8 shadow-lg bg-white rounded-xl overflow-hidden">
        <div className="h-20 bg-gradient-to-br from-purple-200 
             via-rose-200 to-yellow-300" />
        <div className="p-5">
          <p className="text-2xl font-bold text-slate-800">
            {username || "이름 없음"}
          </p>
          <p className="text-gray-500 text-sm">📞 {phone}</p>
          <p className="text-gray-500 text-sm">✉️ {email}</p>
        </div>
      </div>
    </div>
  );
}

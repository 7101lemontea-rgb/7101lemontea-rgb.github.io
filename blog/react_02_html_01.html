01

HTML이란?

HTML(HyperText Markup Language)은 웹 페이지의 뼈대를 구성하는 가장 기본적인 프로그래밍 언어(정확히는 마크업 언어)입니다.

HyperText (하이퍼텍스트) : 링크를 클릭해 이 페이지에서 저 페이지로 자유롭게 이동할 수 있게 해주는 기능입니다.

Markup Language (마크업 언어) : 문서의 구조를 표시하기 위해 태그(Tag)라는 특별한 기호를 사용하는 언어입니다.

HTML은 꺾쇠 괄호(< >)로 둘러싸인 '태그'를 사용해 웹페이지의 요소들을 정의합니다. 보통 시작 태그와 종료 태그가 한 쌍으로 이루어져 있습니다.

HTML
<h1>이것은 큰 제목입니다</h1>
<p>이것은 하나의 문단(단락)입니다.</p>
<img src="image.jpg">  <!-- 종료 태그가 없는 빈 태그 -->

02

HTML 기본 구조

1.1. HTML 기본 구조

HTML 문서는 크게 4가지 핵심 요소로 구성됩니다.

<!DOCTYPE html> HTML5 문서임을 브라우저에 선언 (최신 표준 모드로 렌더링하도록 지시)
<html> HTML 문서의 루트(최상위) 요소
<head> 메타데이터 영역 (문자셋, 타이틀, CSS/JS 링크 등)
<body> 실제 화면에 표시되는 콘텐츠 영역
HTML index.html 
<!DOCTYPE html>
<html lang="ko">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>웹페이지 제목</title>
  </head>
  <body>
    <!-- 실제 화면에 보이는 내용 -->
  </body>
</html>

1.2. HTML 구성요소

구성요소 설명 예시
태그 (tag) HTML 요소를 정의하는 데 사용되는 코드. 시작태그, 종료태그가 있음. <p></p>, <img src="img.jpg">, <br>
요소 (element) 태그와 태그 사이에 있는 내용을 합쳐서 '요소'라고 부른다. 내용을 갖는 요소, 내용을 갖지 않는 요소가 있음. <p>hello</p>, <img src="img.jpg">
속성 (attribute) 요소에 추가적인 정보를 제공하는 부분. 열린 태그에서 정의. (속성이름=속성값) <img src="img.png">
내용 (content) 시작태그와 종료태그 사이에 위치한 내용. 내용을 갖는 태그, 갖지 않는 태그가 있음. <p>hello</p>

2.1. 제목

h1이 가장 크고 h6이 가장 작다.

HTML
<h1>가장 중요한 제목 (h1)</h1>
<h2>두 번째 제목 (h2)</h2>
<h3>세 번째 제목 (h3)</h3>
<h4>네 번째 제목 (h4)</h4>
<h5>다섯 번째 제목 (h5)</h5>
<h6>가장 작은 제목 (h6)</h6>

2.2. 텍스트

p(paragraph) — 문단을 나타내는 태그입니다. 자동으로 위아래 여백이 생깁니다. 각 p 태그는 독립된 문단으로 구분됩니다.

HTML
<p>문단을 나타내는 태그입니다. 자동으로 위아래 여백이 생깁니다.</p>
<p>각 p 태그는 독립된 문단으로 구분됩니다.</p>

br — 줄바꿈 태그입니다.

HTML
<p>첫 번째 줄<br>두 번째 줄</p>

2.3. 링크

a(anchor) — 하이퍼링크를 만드는 태그입니다.

HTML
<!-- 외부 링크 -->
<a href="https://www.google.com">구글로 이동</a>

<!-- 새 탭에서 열기 -->
<a href="https://www.naver.com" target="_blank">네이버 (새 탭)</a>

<!-- 같은 페이지 내 이동 -->
<a href="#section2">섹션2로 이동</a>

<a href="주소"> : 하이퍼링크    target="_blank" 속성 사용 시 새 탭에서 열립니다.

2.4. 이미지

img(image) — 이미지를 삽입하는 빈 태그입니다. alt는 이미지가 안 보일 때 대신 표시되는 텍스트이자 시각장애인을 위한 설명입니다.

HTML
<!-- 기본 이미지 -->
<img src="photo.jpg" alt="사진 설명">

<!-- 크기 지정 -->
<img src="logo.png" alt="로고" width="200" height="100">

<!-- 온라인 이미지 -->
<img src="https://via.placeholder.com/300x200" alt="placeholder">

<img src="이미지경로" alt="대체텍스트">
alt 는 이미지가 안 보일 때 대신 뜨는 텍스트이자 시각장애인을 위한 설명입니다.

2.5. 레이아웃

div(division) — 블록 레벨 컨테이너(한 줄 전체 차지)

HTML
<div style="background-color: lightblue; padding: 20px;">
  <p>div는 여러 요소를 그룹화하는 컨테이너입니다.</p>
  <p>블록 레벨이므로 새 줄에서 시작합니다.</p>
</div>

span — 인라인 컨테이너(내용만큼 차지)

HTML
<p>
  이 문장에서 <span style="color: red; font-weight: bold;">빨간색 굵은 글씨</span>만
  강조하고 싶을 때 span을 사용합니다.
</p>

2.6. 목록

ul, ol, li — 목록을 만드는 태그입니다.

<ul> : 순서 없는 목록 (점 표기)    <ol> : 순서 있는 목록 (1, 2, 3…)    <li> : 목록의 각 항목

HTML
<!-- 순서 없는 목록 -->
<ul>
  <li>첫 번째 항목</li>
  <li>두 번째 항목</li>
  <li>세 번째 항목</li>
</ul>

<!-- 순서 있는 목록 -->
<ol>
  <li>첫 번째 단계</li>
  <li>두 번째 단계</li>
  <li>세 번째 단계</li>
</ol>

04

인라인 요소 vs 블록레벨 요소

인라인 요소

문서의 흐름을 끊지 않고, 한 줄에 표시(줄바꿈 안됨)

웹페이지의 내용을 구성하거나 링크를 추가하는 등의 작업에 사용

<span> <a> <img> <audio> <video> <object> <embed> <strong> <em> <td> <th>
블록레벨 요소

새로운 줄에서 시작 (줄바꿈을 동반함)

웹페이지의 구조와 레이아웃을 정의하는 데 주로 사용

<div> <p> <hn> <ul> <ol> <li> <dl> <dt> <dd> <table> <tr>

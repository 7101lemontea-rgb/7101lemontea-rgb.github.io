Tailwind CSS

Tailwind는 별도의 .css 파일 없이 HTML class 속성에 단어만 적으면 디자인이 완성되는 유틸리티 퍼스트 CSS 프레임워크입니다. Tailwind의 핵심 규칙부터 자주 쓰는 클래스까지 정리합니다.

Section 09

Tailwind CSS란?

  • 미리 만들어놓은 CSS 클래스 모음집입니다.
  • 별도의 .css 파일을 만들어서 클래스 이름을 고민할 필요 없이, HTML 태그의 class 속성에 미리 정의된 단어들을 적기만 하면 디자인이 완성됩니다.
기존 CSS 방식

HTML과 CSS 파일을 따로 작성합니다.

HTML 
<div class="card">안녕하세요</div>
CSS 
.card {
  padding:          16px;
  margin:           8px;
  background-color: white;
  border-radius:    8px;
}
Tailwind 방식

CSS 파일을 따로 안 만들고, HTML의 class에 직접 스타일을 적는 방식입니다.

HTML 
<div class="p-4 m-2 bg-white rounded-lg">
  안녕하세요
</div>
Section 09 — 1

핵심 규칙

Tailwind의 숫자는 0.25rem (4px) 단위입니다.

숫자 실제 크기
1 4px (0.25rem)
2 8px (0.5rem)
4 16px (1rem)
8 32px (2rem)
16 64px (4rem)
Section 09 — 2

자주 쓰는 기초 단어

a. 색상 (Color)

  • text- — 글자 색상 ( text-blue-600 )
  • bg- — 배경 색상 ( bg-gray-100 )
  • border- — 테두리 색상 ( border-black )
  • 숫자(100~900)가 커질수록 색이 진해집니다.

b. 박스모델 (간격과 크기)

m (margin) — 외곽 여백

Tailwind CSS
m-4 margin: 1rem;
mt-4 margin-top: 1rem;
mb-4 margin-bottom: 1rem;
ml-4 margin-left: 1rem;
mr-4 margin-right: 1rem;
mx-4 (horizontal) margin-left: 1rem; margin-right: 1rem; — 좌우 한꺼번에
my-4 (vertical) margin-top: 1rem; margin-bottom: 1rem; — 상하 한꺼번에

p (padding) — 내부 여백 ( p-5, px-3 등 )

padding도 margin과 동일한 방식으로 pt-, pb-, pl-, pr-, px-, py-를 사용합니다.

w / h — 너비와 높이 ( w-full, w-20, h-screen )

Tailwind CSS
w-20 width: 5rem;
w-full width: 100%; — 부모 요소 너비의 100%
w-1/2 width: 50%;
h-screen height: 1vh; — 현재 보고 있는 화면 높이 전체를 채움

c. Flexbox

Tailwind CSS
flex display: flex;
flex-col flex-direction: column;
items-center align-items: center;
justify-center justify-content: center;
justify-between justify-content: space-between;
gap-4 gap: 16px;

d. 텍스트

Tailwind CSS
text-lg font-size: 18px; — 큰 글씨
text-sm font-size: 14px; — 작은 글씨
font-bold font-weight: bold;
text-center text-align: center;

폰트 크기

클래스 픽셀(px) 비고
text-xs 12px 아주 작은 주석, 캡션
text-sm 14px 부가 설명, 작은 본문
text-base 16px 기본값 (대부분의 본문)
text-lg 18px 강조하고 싶은 본문, 소제목
text-xl 20px 게시글 제목
text-2xl 24px 큰 제목
text-4xl 36px 메인 타이틀 (Hero 섹션)

e. 배경 / 테두리

Tailwind CSS
rounded-lg border-radius: 8px;
border border: 1px solid;
shadow-md 중간 크기 그림자 효과

둥글기 정도

rounded-sm

아주 살짝 둥글게 (2px)

rounded

기본 둥글게 (4px)

rounded-md

조금 더 둥글게 (6px)

rounded-lg

확연히 둥글게 (8px)

rounded-xl

많이 둥글게 (12px)

rounded-full

완전히 둥글게 — 알약 모양이나 원형 만들 때 사용

f. 색상 체계

Tailwind 색상은 이름 + 밝기(100~900) 조합으로 사용합니다.

색상명-숫자 형태로 작성합니다. 숫자가 클수록 색이 진해집니다.

밝기 느낌 사용 예
100 가장 연함 (거의 흰색) 배경, hover 효과
300 ~ 400 연함 비활성 버튼, 보조 텍스트
500 기본 (표준 색상) 기본 버튼, 아이콘
600 ~ 700 진함 주요 텍스트, 강조 버튼
900 가장 진함 (거의 검정) 제목, 본문 텍스트

g. hover

  • 클래스 앞에 hover: 만 붙이면 됩니다.
HTML
<div class="bg-blue-500 hover:bg-blue-700"></div>
→ 평소엔 파란색, 마우스를 올리면 진한 파란색

h. 반응형

Tailwind의 킬러 기능. 접두사 하나로 미디어쿼리 끝.

접두사 의미 (최소 너비) 실제 장치 느낌
(없음) 0px ~ 모바일 (기본)
sm: 640px ~ 큰 스마트폰 / 가로모드
md: 768px ~ 태블릿 (iPad 등)
lg: 1024px ~ 노트북 / 데스크탑
xl: 1280px ~ 큰 모니터
HTML — 반응형 텍스트 크기 예시 
<div class="text-sm md:text-lg lg:text-2xl">
  모바일에선 작게, 태블릿에선 크게, 데스크탑에선 더 크게
</div>

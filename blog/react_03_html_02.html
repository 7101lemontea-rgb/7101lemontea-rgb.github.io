5. 폼(Form) 요소

로그인, 회원가입, 검색창까지. 사용자 데이터를 서버로 전송하는 모든 흐름의 시작, <form> 태그를 정리합니다.

5.1

폼(Form)

HTML에서 <form>은 사용자가 입력한 데이터를 서버로 전송하기 위한 입력 영역입니다.

로그인, 회원가입, 검색창, 설문조사 같은 기능이 모두 form을 사용합니다. 사용자가 입력한 데이터를 서버(백엔드)로 전송하기 위해 <form> 태그가 존재합니다.

HTML 
<form action="서버주소" method="전송방식">
  <!-- 입력 요소들 -->
</form>
action

데이터를 보낼 목적지 URL
예 /login, /search

method

데이터를 보내는 방법.
GET · POST

GET vs POST

항목 GET POST
용도 데이터 조회 데이터 저장
보안 낮음 상대적으로 높음
길이 제한 있음 거의 없음
캐싱 가능 불가능
5.2

주요 입력 태그와 속성

input

가장 많이 쓰이며 type 속성에 따라 모습이 바뀝니다.

HTML
<!-- 텍스트 입력 -->
<input type="text"     placeholder="이름을 입력하세요">
<input type="password" placeholder="비밀번호">
<input type="email"    placeholder="이메일 주소">

<!-- 숫자 입력 -->
<label> 출생연도
  <input type="number" min="1900" max="2025" value="2000">
</label>

<!-- 날짜 입력 -->
<input type="date" name="birthday">

<!-- 체크박스(같은 name으로 그룹화 - 복수선택) -->
<label><input type="checkbox" name="interests" value="sports"/>스포츠</label>
<label><input type="checkbox" name="interests" value="music" />음악</label>
<label><input type="checkbox" name="interests" value="travel"/>여행</label>

<!-- 라디오버튼(같은 name으로 그룹화 - 단일선택) -->
<label><input type="radio" name="member-type" value="personal"/>개인회원</label>
<label><input type="radio" name="member-type" value="company" />사업자회원</label>
type 값 설명
text 일반 텍스트 (이름, 아이디)
password 비밀번호 (글자가 가려짐)
email 이메일 형식이 아니면 경고 (HTML5)
number 숫자만 입력
date 날짜 선택 달력 표시 (HTML5)
checkbox 여러 개 중복 선택 (취미, 옵션 추가)
radio 여러 개 중 하나만 선택 (동의/비동의)

label

HTML
<!-- id와 for 값을 일치시켜야 연결됨 -->
<label for="username">이름:</label>
<input type="text" id="username">

textarea

자기소개, 가입 인사, 배송 요청 사항처럼 여러 줄의 긴 글을 입력받을 때 사용합니다.

HTML
<textarea
  id="req_memo"
  name="req_memo"
  rows="5"
  cols="40"
  maxlength="200"
  placeholder="배송 요청 사항을 적어주세요.(200자 이내)"
></textarea>

textarea는 value 속성이 없습니다. 여는 태그와 닫는 태그 사이에 내용을 작성합니다.

❌ 틀린 방법

<textarea value="안녕하세요"></textarea>

✅ 맞는 방법

<textarea>안녕하세요</textarea>

select · option

드롭다운 메뉴를 만들 때 사용하며, <select>라는 큰 틀 안에 선택지 하나하나를 <option> 태그로 넣습니다.

HTML
<!-- 셀렉트 박스 -->
<select name="menu">
  <option value="">선택하세요</option>
  <option value="americano">아메리카노</option>
  <option value="cafe_latte" selected>카페라떼</option>
  <option value="ice_tea">아이스티</option>
</select>
속성 설명
value 사용자가 선택했을 때 서버에 전송되는 값
selected 페이지가 열릴 때 기본으로 선택되어 있을 항목에 붙여줍니다
5.3

버튼

HTML
<button type="button">클릭하세요</button>
<button type="submit">제출</button>
<button type="reset">초기화</button>
submit type

폼의 내용(입력값)을 서버로 전송 — 가장 중요한 버튼 타입

button type

자바스크립트로 기능을 붙일 때 사용

reset type

입력한 내용을 전부 지우고 초기화

5.4

유효성 검사를 위한 속성

자바스크립트 없이도 기본적인 검사가 가능합니다.

속성 설명
placeholder="..." 입력 전 힌트 메시지 표시
required 필수 입력 항목 지정 (비어있으면 제출 안 됨)
readonly 읽기 전용 (수정 불가)
disabled 비활성화 (전송도 안 됨)
name 서버로 전송될 때 사용되는 이름
value 초깃값 또는 사용자 입력값

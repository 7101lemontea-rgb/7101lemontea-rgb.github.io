01

Javascript 실행 환경

HTML 파일 내에서 JavaScript를 실행하는 방법은 크게 두 가지로 나뉩니다. HTML 태그 사이에 직접 코드를 작성하거나, 외부 .js 파일을 불러오는 방식입니다.

1.1. HTML 파일 내에서 실행

HTML 파일의 또는 태그 안에 직접 자바스크립트 코드를 작성하는 방식입니다. 간단한 동적 기능을 구현할 때 주로 사용합니다.

html 
<!DOCTYPE html>
<html>
<head>
  <title>JS 연습</title>
</head>
<body>
  <h1>JavaScript 시작!</h1>
  <!-- body 닫히기 직전에 위치 권장 -->
  <script>
    console.log("안녕하세요!");
    alert("환영합니다!");
  </script>
</body>
</html>

1.2. 외부 파일로 분리

자바스크립트 코드를 별도의 .js 파일로 저장하고, HTML에서 경로를 지정해 불러오는 방식입니다. 코드의 재사용성이 높고 HTML 구조와 로직을 분리할 수 있습니다.

html index.html 
<!-- index.html -->
<!DOCTYPE html>
<html>
<head>
  <title>JS 연습</title>
</head>
<body>
  <h1>JavaScript 시작!</h1>
  <script src="script.js"></script>
</body>
</html>
js script.js 
// script.js
console.log("외부 파일에서 실행!");

1.3. 브라우저 개발자 도구

  • F12 → Console 탭에서 바로 실행해볼 수 있습니다.

개발자 도구 Console 탭에서 직접 입력해보세요:

console 
> 1 + 1
< 2

> "Hello" + " World"
< "Hello World"

02

Javascript 출력 방법

js 
// 1. 콘솔 출력 (개발자 도구에서 확인) - 가장 많이 사용
console.log("콘솔에 출력");

// 2. 알림창
alert("알림창 출력");

03

Javascript 코드 작성 규칙

  • 대소문자를 구분합니다.
  • 중괄호 {} 를 사용하여 코드 블록을 묶어줍니다.
  • 문장의 끝에 세미콜론 ; 을 붙입니다.
  • 주석
    • 한줄 주석 : //
    • 여러줄 주석 : /* */

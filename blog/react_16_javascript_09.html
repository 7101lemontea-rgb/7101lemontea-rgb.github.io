01

템플릿 리터럴

템플릿 리터럴은 내장된 표현식을 허용하는 문자열 리터럴입니다. 기존의 따옴표 대신 백틱(`)을 사용하여 문자열을 더 직관적으로 다룰 수 있습니다.

1.1. 백틱(`) 사용법

백틱을 사용하면 문자열 안에 표현식을 쉽게 삽입할 수 있습니다 (파이썬의 f-string과 유사). 

const str = `이것이 템플릿 리터럴입니다.`;

여러 줄 문자열

줄바꿈(엔터)을 별도의 특수문자 없이 그대로 문자열에 포함할 수 있습니다. 

const text = `여러 줄의
문자열을
그대로 쓸 수 있습니다.`;

변수/표현식 삽입

${변수명} 형태로 값을 쉽게 삽입합니다.

const name = "홍길동";
const age = 25;

// 기존 방식 (+ 로 연결)
const intro1 = "제 이름은 " + name + "이고, " + age + "살입니다.";

// 템플릿 리터럴 (${} 사용)
const intro2 = `제 이름은 ${name}이고, ${age}살입니다.`;

// 표현식 계산도 가능
const total = `총 금액: ${1000 * 3}원`;

※ 연습 문제: 템플릿 리터럴로 자기소개 만들기

const myName = "김코딩";
const myJob = "개발자";
const myAge = 30;

const intro = `저는 ${myAge}살 ${myJob} ${myName}입니다.`;

02

삼항 연산자

삼항 연산자는 조건문에 따라 값을 결정하는 간결한 방법입니다. 조건 ? 참 : 거짓 형태로 사용합니다.

2.1. 한 줄 if문

let score = 70;
let result = score >= 60 ? "합격입니다" : "불합격입니다.";

중첩 사용 (가독성 주의) 

const grade = score >= 90 ? "A" : score >= 80 ? "B" : "C";

[Why React?] 리액트의 JSX 중괄호{ } 안에서는 if문을 직접 사용할 수 없으므로, 조건부 렌더링을 위해 삼항 연산자가 필수적으로 사용됩니다.

연습 문제

숫자를 입력받아 0, 짝수, 홀수를 판별하는 화살표 함수를 삼항 연산자로 구현해보세요.

js
const checkNum = (n) => (n === 0 ? 0 : n % 2 === 0 ? "짝수" : "홀수");

console.log(checkNum(30)); // "짝수"

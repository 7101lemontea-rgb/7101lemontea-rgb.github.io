DOM (Document Object Model)

DOM 개요

DOM(Document Object Model, 문서 객체 모델)은 HTML, XML 문서의 프로그래밍 인터페이스입니다. 브라우저가 정적인 텍스트 파일인 HTML을 읽어 들여, 이를 프로그램이 수정·삭제·추가할 수 있는 객체(Object) 구조로 변환한 것을 의미합니다.

JavaScript로 이 트리의 요소들을 선택하여 요소의 내용, 속성, 스타일을 변경하고, 새로운 요소를 추가하거나 기존 요소를 삭제할 수 있습니다.

HTML 문서 원본 파일

브라우저가 텍스트 파일(.html)을 읽어 들입니다.

DOM 트리 생성 파싱

HTML 요소들을 노드(Node) 객체 트리로 변환합니다.

JavaScript 제어 동적 조작

선택·변경·추가·삭제 등 DOM을 자유롭게 조작합니다.

[참고] 리액트는 DOM을 직접 제어하지 않고, 상태(State)로 화면을 바꾸는 방식을 사용합니다.

핵심 노드 타입

DOM은 HTML 요소들을 노드(Node)라고 불리는 객체들로 표현합니다.

Document
Document 노드

DOM 트리의 최상위 노드입니다.

Element
Element 노드

HTML 태그 — <h1>, <p>, <div> 등 모든 HTML 요소입니다.

Text
Text 노드

HTML 요소 내의 텍스트 콘텐츠입니다.

01

요소 선택하기

주요 DOM 메서드로 원하는 HTML 요소를 선택할 수 있습니다.

메서드 반환값 선택자
getElementById('id') 단일 요소 id로 선택
querySelector('선택자') 단일 요소 CSS 선택자로 첫 번째 요소
querySelectorAll('선택자') NodeList CSS 선택자로 모든 요소

예제

HTML

html
<h1 id="title">안녕하세요</h1>
<p class="desc">첫 번째 설명</p>
<p class="desc">두 번째 설명</p>

단일 요소 선택

js
const title = document.getElementById('title');
const firstDesc = document.querySelector('.desc');

여러 요소 선택

js
const allDesc = document.querySelectorAll('.desc');

console.log(title);          // <h1 id="title">안녕하세요</h1>
console.log(allDesc.length); // 2

02

요소 내용 변경하기

선택한 요소의 내용을 읽거나 바꿀 때 사용하는 두 가지 속성입니다.

속성 설명
textContent 텍스트만 변경 (HTML 태그 무시)
innerHTML HTML 포함 변경 (보안 주의!)

예제

js
const title = document.getElementById('title');

// 텍스트 읽기
console.log(title.textContent); // "안녕하세요"

// 텍스트 변경
title.textContent = '반갑습니다';

// HTML 포함 변경
title.innerHTML = '반갑습니다 <small>😊</small>';
textContent

HTML 태그를 문자 그대로 출력합니다. 더 안전합니다. 

title.textContent = '<b>굵게</b>';
// 화면: <b>굵게</b> (태그가 그대로 보임)
innerHTML

HTML 태그를 실제 요소로 렌더링합니다.

title.innerHTML = '<b>굵게</b>';
// 화면: 굵게 (실제로 굵어짐)

03

이벤트 연결

addEventListener를 사용하면 특정 동작이 발생했을 때 실행할 함수를 요소에 연결할 수 있습니다.

js
element.addEventListener('이벤트타입', 콜백함수);

콜백함수는 괄호를 붙이지 않고 함수명만 전달합니다. 매개변수가 있는 함수를 전달해야 한다면 화살표 함수를 사용합니다.

자주 쓰는 이벤트

이벤트 발생 시점
click 클릭했을 때
input 입력값이 변할 때
submit 폼 제출할 때
keydown 키를 눌렀을 때
mouseover / mouseout 마우스 올림 / 벗어남

예제 1 — 클릭수

HTML

html
<button id="btn">클릭!</button>
<p id="result"></p>

JavaScript

js
const btn = document.getElementById('btn');
const result = document.getElementById('result');
let clickCount = 0;

btn.addEventListener('click', function() {
    clickCount++;
    result.textContent = `${clickCount}번 클릭했습니다`;
});

예제 2 — 실시간 글자수 표시

HTML

html
<textarea id="message" placeholder="메시지 입력"></textarea>
<p>글자수: <span id="count">0</span>/100</p>

JavaScript

js
const message = document.getElementById('message');
const count = document.getElementById('count');

message.addEventListener('input', function() {
    count.textContent = message.value.length;
});

input 이벤트는 키를 누를 때마다 즉시 발생합니다. message.value로 현재 입력된 텍스트를 가져오고, .length로 글자 수를 셉니다.

Overview

분석 개요

본 문서는 세 가지 그룹웨어 도메인(웨딩플래너, 자격증 준비, 꽃 예약)에서 기존 솔루션의 강점과 한계를 분석하고, 각 영역의 시장 공백을 도출합니다. 분석 결과는 신규 서비스 기획의 근거 자료로 활용됩니다.

01 — Wedding Planner Groupware

웨딩플래너 그룹웨어

핵심 관찰: 플래너(B2B)↔예비부부(소비자)가 같은 화면을 보며 소통하는 서비스가 존재하지 않습니다.

기존 솔루션 1 — 아이웨딩 (앱)

연계 업체와 웨딩플래너 매칭을 중심으로 한 소비자용 웨딩 앱입니다.

강점 Strengths
  • 연계된 업체가 많아 원스톱 선택 가능
  • 스튜디오·드레스·메이크업을 결합해 할인 제공 (스드메 패키지)
  • 웨딩플래너를 배정받아 업체 예약을 대행
한계 Limitations
  • 플래너가 제안하는 비용 외 세부 내역을 전부 공개하지 않음 → 비용 불투명
  • 웨딩플래너가 전체 준비 일정을 통합 관리하지 않음 → 일정 공백 발생
  • 커플이 직접 확인·비교할 수 있는 투명한 데이터 화면 없음

기존 솔루션 2 — 다이렉트 결혼준비 (네이버 카페)

대규모 커뮤니티 기반의 정보 공유 플랫폼입니다. 이용자 수가 많아 풍부한 후기가 형성되어 있습니다.

강점 Strengths
  • 연계된 업체가 많고 카페 글 작성 시 할인 혜택 제공
  • 이용자 수가 많아 다양한 업체 후기 수집 가능
  • 커뮤니티 활성화로 예비부부 간 정보 교환 활발
한계 Limitations
  • 연계되지 않은 업체의 정보는 파악 어려움 → 정보 편향
  • 할인을 받으려면 반드시 카페에 글을 써야 하는 조건 강제
  • 부정적 후기는 노출이 제한되어 공정한 정보 제공 불가
  • 개인 일정 관리·플래너 연동 기능 없음

기존 솔루션 3 — 푸딩 (앱)

체크리스트, 청첩장, 하객 관리 등 예비부부의 셀프 준비를 지원하는 앱입니다.

강점 Strengths
  • 결혼 준비 체크리스트 기능 제공
  • 청첩장 제작 및 하객 관리 기능 통합
  • 예비부부가 독립적으로 준비할 수 있는 UI 구성
한계 Limitations
  • 웨딩플래너와의 연계 기능 부재 → 플래너 사용 커플에게 부적합
  • 웨딩업체와의 직접 연동 미흡 → 업체 정보 확인 불편
  • 비용 명세서 공유·실시간 소통 기능 없음

기존 솔루션 4 — 웨드소프트 (웨딩플래너용 B2B)

웨딩플래너와 웨딩업체를 위한 전용 B2B 솔루션입니다. 고객관리·발주서·일정·정산 관리 및 업종별 특화 모듈을 제공합니다.

강점 Strengths
  • 웨딩플래너의 고객관리, 발주서, 일정, 정산 관리 통합 제공
  • 웨딩홀·스튜디오·드레스샵·메이크업샵 각각에 특화된 업무 모듈
  • 플래너와 업체 입장에서 업무 효율 극대화
한계 Limitations
  • 신혼부부(소비자)는 플래너가 관리하는 내용을 전혀 볼 수 없음
  • 플래너 중심의 단방향 시스템 → 커플과의 협업 화면 없음
  • 비용 명세, 일정, 계약 현황 등을 커플과 공유하는 기능 부재

시장 공백

시장 공백: 플래너(B2B)와 예비부부(소비자)가 같은 화면을 보며 비용·일정·채팅을 함께 관리하는 서비스가 존재하지 않습니다.

기존 4개 솔루션의 공통 한계를 종합하면, 다음 세 가지 영역이 시장에서 완전히 비어 있습니다.

  • 비용 투명화: 플래너가 입력한 비용 명세를 커플이 실시간으로 확인·비교할 수 있는 화면
  • 일정 공유: 플래너가 등록한 준비 단계별 일정을 커플이 함께 보고 D-day를 관리하는 기능
  • 실시간 협업 채팅: 플래너와 커플이 같은 워크스페이스에서 문서·업체 정보를 공유하며 소통하는 채널
02 — License Exam Groupware

자격증 준비 그룹웨어

핵심 관찰: 문제풀이, 학습 시간 관리, 커뮤니티가 각각 다른 앱에 분산되어 있으며, 세 기능을 통합한 서비스가 존재하지 않습니다.

기존 솔루션 1 — 맞추다 (웹)

자격증 기출문제를 개인 맞춤 CBT(컴퓨터 기반 시험) 방식으로 제공하는 에듀테크 서비스입니다.

강점 Strengths
  • 자격증 기출문제를 개인 맞춤형 CBT 환경으로 제공
  • 취약점 보완을 위한 오답노트 자동 생성
  • 개인별 학습 패턴 분석 및 피드백 제공
한계 Limitations
  • 커뮤니티 기능이 미약하여 학습자 간 교류·동기부여 기반 부족
  • 학습 시간 관리(타이머, 스톱워치) 기능 없음
  • 스터디 그룹 구성이나 함께 공부하는 기능 부재
  • 모바일 앱 없이 웹 중심 → 접근성 제한

기존 솔루션 2 — 열품타 (앱)

타이머로 공부 시간을 측정하고 그룹·카테고리 내 사람들과 학습 현황을 공유하는 학습 소셜 앱입니다.

강점 Strengths
  • 타이머로 자신의 공부 시간을 정확하게 측정
  • 가입한 그룹 또는 같은 카테고리 사람들에게 학습 시간 공유
  • 랭킹 기능으로 학습 동기부여 강화
  • 그룹 내 친목·응원 커뮤니티 활성화
한계 Limitations
  • 자격증 기출문제 등 문제 관련 기능 전혀 없음
  • 오답노트, 취약점 분석 등 학습 콘텐츠 부재
  • 시간을 재는 것 외에 실질적인 학습 내용 지원 없음
  • 자격증 특화 커리큘럼이나 일정 관리 기능 없음

시장 공백

시장 공백: 기출문제 풀이(맞추다) + 학습 시간 관리(열품타) + 커뮤니티를 하나의 서비스로 통합한 자격증 특화 그룹웨어가 존재하지 않습니다.

  • 문제풀이와 학습 시간을 함께 기록·분석하는 통합 대시보드
  • 자격증 시험 일정을 기준으로 역산한 스터디 플랜 자동 생성
  • 같은 자격증을 준비하는 사람들과 학습 현황을 공유하는 스터디 그룹 기능
  • 오답노트와 학습 시간 데이터를 연결한 취약점 집중 학습 추천
03 — Flower Reservation Groupware

꽃 예약 그룹웨어

핵심 관찰: 기존 플랫폼은 소비자 주문 중계에만 집중하며, 개별 꽃집의 내부 업무관리(재고, 주문, 배달, 정산)를 지원하는 도구가 전무합니다.

기존 솔루션 1 — 꽃집청년들 · 플라워119

소비자와 가맹 꽃집을 연결하는 꽃 배달 O2O 플랫폼입니다.

강점 Strengths
  • 소비자 주문을 가맹 꽃집에 연결하는 중계 플랫폼으로 인지도 확보
  • 꽃 배달 특화로 빠른 주문·결제 프로세스 제공
  • 전국 가맹점 네트워크 확보로 커버리지 넓음
  • 소비자 앱 UX가 직관적으로 설계됨
한계 Limitations
  • 개별 꽃집 내부 업무관리(재고, 주문 현황, 직원 관리) 기능 전혀 없음
  • 주문 접수 후 제작·포장·배달 상태를 내부에서 관리하는 도구 부재
  • 꽃의 입고·신선도·재고 현황을 실시간 추적하는 기능 없음
  • 가맹점이 아닌 개인 꽃집은 플랫폼 이용 자체가 어려움
  • 매출 정산, 수익 분석, 월별 보고 등 경영 지원 기능 전무
  • 예약 주문(웨딩, 행사용 꽃다발)과 일반 배달 주문을 구분 관리하는 기능 없음

꽃집 내부 업무 흐름 분석 (미지원 영역)

기존 솔루션이 전혀 다루지 않는 꽃집 내부 업무 흐름을 단계별로 정리합니다. 이 흐름 전체가 시장 공백 영역입니다.

01
재고 입고 Inventory

업무 내용: 꽃 품종별 수량, 입고일, 신선도 기한 기록

미지원 시 문제점: 수기 관리로 오류 발생, 신선도 초과 폐기 증가

02
주문 접수 Order

업무 내용: 전화·SNS·앱 등 다채널 주문을 한 곳에서 확인

미지원 시 문제점: 채널별 분산 관리로 누락·중복 주문 발생

03
제작 배정 Production

업무 내용: 직원별 제작 담당 배정, 포장 완료 상태 추적

미지원 시 문제점: 구두 전달로 인한 혼선, 제작 현황 파악 불가

04
배달 관리 Delivery

업무 내용: 배달 지역·시간·담당자 배정 및 완료 확인

미지원 시 문제점: 배달 상태 실시간 추적 불가, 고객 문의 응대 어려움

05
예약 주문 Reservation

업무 내용: 웨딩·행사용 대량 주문의 날짜별 사전 관리

미지원 시 문제점: 일반 주문과 혼재되어 누락 위험 높음

06
정산·매출 Settlement

업무 내용: 일별·월별 매출 집계, 비용 대비 수익 분석

미지원 시 문제점: 수기 엑셀 관리로 오류 많고 시간 소모 큼

시장 공백

시장 공백: 소비자 주문 중계가 아닌, 꽃집 사장과 직원이 재고·주문·제작·배달·정산을 한 화면에서 관리하는 꽃집 전용 그룹웨어가 존재하지 않습니다.

  • 재고 관리: 품종별 입고 수량, 신선도 만료일 추적, 재주문 알림
  • 주문 통합 관리: 전화·SNS·앱 등 다채널 주문을 하나의 대시보드에서 확인
  • 제작·배달 현황 추적: 주문별 제작 상태(대기/제작중/완료/배달중)를 실시간으로 관리
  • 예약 주문 분리: 웨딩·행사용 대량 예약을 일반 배달 주문과 별도로 관리하는 캘린더
  • 정산 자동화: 일별·월별 매출 자동 집계 및 수익 분석 리포트
Conclusion

종합 결론 — 시장 공백 요약

Domain 01 웨딩플래너 그룹웨어
핵심 시장 공백
플래너와 예비부부가 같은 화면에서 비용·일정·채팅을 함께 보는 서비스가 없음
신규 서비스 기회
플래너↔커플 공유 워크스페이스: 비용 투명화 + 일정 공유 + 실시간 협업 채팅
Domain 02 자격증 준비 그룹웨어
핵심 시장 공백
문제풀이(맞추다) + 학습 시간(열품타) + 커뮤니티를 통합한 서비스가 없음
신규 서비스 기회
자격증 특화 올인원 앱: 기출풀이 + 타이머 + 스터디 그룹 + 취약점 분석
Domain 03 꽃 예약 그룹웨어
핵심 시장 공백
꽃집 내부 업무(재고·주문·제작·배달·정산)를 통합 관리하는 도구가 없음
신규 서비스 기회
꽃집 전용 그룹웨어: 재고 추적 + 주문 통합 + 배달 현황 + 매출 정산

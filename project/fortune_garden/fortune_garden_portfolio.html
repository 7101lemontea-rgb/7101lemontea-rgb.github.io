Project — Fortune Garden
2인 가구를 위한 가계부 앱

엑셀로 관리하던 가계부를 앱으로 전환한 개인 프로젝트입니다. 2인 뷰 분리로 개인/통합 지출을 필터 없이 바로 확인하고, 월별 차트가 데이터와 연동되어 자동으로 그려집니다.
Windows · Android 크로스플랫폼으로 월별 리포트와 카테고리 자동 분류를 지원합니다.

Retrospective
Problem
CODEF API로 실제 계좌를 연동하려 했으나, 사용하려면 사업자 등록 및 정식 허가가 필요해 개인 프로젝트에서는 적용이 불가능했습니다.
Action
계좌 직접 연동 대신 은행 앱에서 내보낸 CSV 파일을 가져오는 방식으로 전환하고, 기관별 CSV 파싱 프로파일과 카테고리 자동 분류 규칙을 구현했습니다.
Insight
외부 API 도입 전 인증/허가 요건을 먼저 검토해야 한다는 것을 깨달았습니다. 그리고 제약이 생겼을 때 대안 설계로 빠르게 전환하는 것도 중요하다는 것을 경험했습니다.
핵심 기능
개인/통합 뷰 월별 리포트 카테고리 자동 분류 PIN 잠금 DB 백업/복원 CSV 가져오기

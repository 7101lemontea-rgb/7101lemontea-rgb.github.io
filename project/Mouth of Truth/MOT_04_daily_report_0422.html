04 — Key Discussion

주요 논의 및 이슈

논의 1: 화면 흐름 개선 (UX 최적화)

사용자 피로도를 줄이기 위해 링크 입력 방식에서 지도 기반 검색 방식으로 전환을 결정했습니다.

기존 방식
네이버 지도 링크 직접 복사/붙여넣기 (UX 마찰 발생)
개선 방식
카카오맵 API 연동 → 지도 내 마커 클릭 → 즉시 분석 시작 (직관성 강화)

논의 2: 유사 서비스 차별화 분석

  • AD Finder: 블로그 광고 탐지 연구이나 상용화 사례 미비
  • AWOL: 인증된 리뷰만 허용하는 방식으로 서비스 성격이 다름
  • 차별점: 네이버 블로그 리뷰를 실시간 필터링하는 일반 소비자용 B2C 서비스 공백 확인

이슈 / 특이사항

Crawl-Block

네이버 크롤링 차단 가능성 대비 딜레이 로직 및 우회 방안 검토 필요

Detection Accuracy

협찬 문구 없는 지능형 광고 탐지를 위한 문체 분석 가중치 설계 필요

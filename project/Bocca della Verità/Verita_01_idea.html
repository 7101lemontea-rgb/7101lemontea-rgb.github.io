02 — Problem Domain

우리가 주목한 문제 영역

신뢰를 잃은 리뷰 데이터

최근 맛집 소비 결정의 핵심 정보원인 네이버 지도 리뷰가 '바이럴 마케팅'과 '대가성 리뷰'로 인해 신뢰도를 잃고 있으며, 사용자는 극심한 '리뷰 피로도'를 겪고 있습니다.

  • 정보 과부하: 진짜 후기를 찾기 위해 광고 표기 여부를 일일이 확인해야 하는 수고로움 발생
  • 박탈감 유발: 광고에 속아 시간과 비용을 낭비했을 때 느끼는 사용자 경험 저하
  • 공통 경험: 팀원 모두가 식당 탐색 시 광고를 걸러내기 위해 과도한 시간을 소요하는 페인 포인트를 공유
03 — User Analysis

사용자 정의 및 페인 포인트

User 1 신중한 맛집 탐색러
중요한 데이트나 모임 장소를 위해 냉정한 실제 평가를 원하는 2030 사회초년생
User 2 여행자 및 로컬 탐방객
홍보용 블로그가 아닌 진짜 현지인 맛집 정보를 빠르게 식별하고 싶은 방문자

현재 사용자의 불편 (Pain Point)

01
정보의 불균형
광고 자본을 많이 쓴 가게만 검색 상단에 노출되어 찐 맛집이 가려짐
02
높은 검색 비용
식당 한 곳의 진실 여부를 판단하기 위해 5개 이상의 블로그를 교차 검증
03
진화하는 가짜 후기
이미지로 처리된 대가성 문구 등 교묘하게 검색 엔진을 교란하는 사례 증가
04 — Hypothesis & Solution

가설 및 해결 방향

문제 가설 Problem
넘쳐나는 바이럴 마케팅 광고 리뷰는 사용자에게 막대한 시간 낭비와 스트레스를 주며, 이는 잘못된 선택과 서비스 불신으로 이어진다.
해결 가설 Solution
문체 분석과 대가성 표기 위치를 통해 '광고 농도'를 산출하고, 필터링을 통과한 진짜 리뷰만 요약 제공함으로써 신뢰 기반의 미식 생활을 지원한다.
Conclusion

최종 선택 근거 및 다음 단계

공감도 및 수요: 광고 맛집이 아닌 진짜 맛집을 찾고자 하는 대중적 수요가 매우 높습니다.
실현 가능성: 크롤링 기술과 NLP 모델링을 활용하여 MVP 단계에서 구현 가능한 범위임을 확인하였습니다.
학습 가치: 자연어 처리(NLP), 데이터 시각화 기술을 조화롭게 사용하며 실제 페인 포인트를 해결하는 경험을 얻을 수 있습니다.

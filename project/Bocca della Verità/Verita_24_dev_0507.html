3.2. 전처리 함수

def preprocess(text: str) -> str:
    text = unescape(text)                                          # HTML 엔티티 디코딩
    text = re.sub(r"<[^>]+>", "", text)                           # HTML 태그 제거
    text = re.sub(r"#(\w+)", r"\1 ", text)                        # 해시태그 단어 추출
    text = re.sub(r"\.{2,}", " ", text)                           # 말줄임 제거
    text = re.sub(r"[^\w\s가-힣ㄱ-ㅎㅏ-ㅣa-zA-Z0-9.,!?~]", " ", text)  # 특수문자 정리
    text = re.sub(r"\s+", " ", text).strip()
    return text

3.4. WeightedTrainer (가중 손실 함수)

class WeightedTrainer(Trainer):
    def compute_loss(self, model, inputs, return_outputs=False, **kwargs):
        labels  = inputs.pop("labels")
        outputs = model(**inputs)
        loss_fn = CrossEntropyLoss(weight=self.class_weights_tensor.to(model.device))
        loss    = loss_fn(outputs.logits, labels)
        return (loss, outputs) if return_outputs else loss

08. 이슈: 신규 테스트 데이터 성능 저하

8.1. 문제 현상

  • 학습 데이터(val) AUC ~0.95 → 신규 테스트 AUC 0.61로 급락
  • 예측 분포 자체는 정상 (비광고 105건, 광고 145건)

8.2. 원인 분석 — 학습 데이터의 가게명 편향

구분 패턴
광고 샘플 "까몬" 관련 글에 집중
비광고 샘플 "고양이부엌" 관련 글에 집중
결과 모델이 문체/패턴이 아닌 가게명을 판단 기준으로 학습

8.3. 개선 방향

방향 1 — 학습 데이터 다양화

  • 더 다양한 가게의 광고/비광고 샘플 추가
  • 편향된 가게 샘플은 마스킹 버전으로 증강

방향 2 — 가게명 마스킹

가게명을 [STORE] 토큰으로 치환하여 모델의 가게명 의존도를 제거합니다. 

def mask_store_names(text: str, store_vocab: set) -> str:
    words = text.split()
    masked = []
    for word in words:
        clean = re.sub(r"[.,!?~]", "", word)
        if clean in store_vocab:
            masked.append("[STORE]")
        else:
            masked.append(word)
    return " ".join(masked)

09. Supabase 테스트 데이터 업로드 가이드

네이버 API로 수집한 광고 150건 + 비광고 150건을 Supabase에 업로드하는 절차입니다.

Step 1. 테이블 생성 (소문자 테이블명 사용) 

CREATE TABLE public.naverapitest (
  id bigint GENERATED BY DEFAULT AS IDENTITY NOT NULL,
  name text NULL,
  review_url text NULL,
  review_title text NULL,
  review_description text NULL,
  review_bloggername text NULL,
  review_postdate text NULL,
  created_at timestamp with time zone NULL DEFAULT now(),
  is_ad bigint NULL,
  is_ad_bert_pred double precision NULL,
  CONSTRAINT naverapitest_pkey PRIMARY KEY (id)
) TABLESPACE pg_default;

※ PostgreSQL은 따옴표 없이 테이블명을 참조하면 자동으로 소문자 변환하므로, 처음부터 소문자로 생성할 것.

Step 2. 여러 JSON 파일을 하나의 CSV로 병합 

import json, csv, glob

json_files = glob.glob('*.json')
all_data = []

for file in json_files:
    with open(file, 'r', encoding='utf-8') as f:
        json_data = json.load(f)
    if isinstance(json_data, dict):
        json_data = [json_data]
    all_data.extend(json_data)
    print(f"{file} → {len(json_data)}건 추가")

with open('merged.csv', 'w', newline='', encoding='utf-8-sig') as f:
    writer = csv.DictWriter(f, fieldnames=all_data[0].keys())
    writer.writeheader()
    writer.writerows(all_data)

print(f"\n변환 완료: merged.csv ({len(all_data)}건 총합)")

Step 3. 컬럼명을 테이블 스키마에 맞게 변경

CSV 컬럼명 테이블 컬럼명
url review_url
title review_title
description review_description
bloggername review_bloggername
postdate review_postdate 
import pandas as pd

df = pd.read_csv('merged.csv')
df = df.rename(columns={
    'url': 'review_url',
    'title': 'review_title',
    'description': 'review_description',
    'bloggername': 'review_bloggername',
    'postdate': 'review_postdate'
})
df.to_csv('merged_fixed.csv', index=False, encoding='utf-8-sig')
print("완료:", df.columns.tolist())

01. Playwright 크롤링 이슈

1.1. 단축 URL 리다이렉트 처리

네이버 지도 단축 URL(naver.me) 접근 시 실제 콘텐츠가 로드되지 않는 현상이 발생했습니다. 

# 해결: requests를 이용한 리다이렉트 추적
def resolve_short_url(short_url: str) -> str:
    response = requests.get(short_url, allow_redirects=True)
    return response.url

1.2. iframe 내부 요소 탐색 (Locator Timeout)

네이버 지도는 pcmap.place.naver.com iframe 내부에 콘텐츠가 존재하여 메인 DOM 탐색 시 타임아웃이 발생합니다.

  • 원인: iframe 컨텍스트 분리
  • 해결: page.frames를 순회하여 타겟 프레임을 직접 찾아 클릭 동작 수행

1.3. 네트워크 대기 전략 수정

networkidle 설정은 지속적인 통신이 발생하는 페이지에서 무한 대기에 빠질 수 있습니다. 

// 변경 전: wait_until="networkidle"
page.goto(url, wait_until="domcontentloaded")
page.wait_for_timeout(3000) // 수동 대기 시간 확보

02. 네이버 검색 API 이슈

API 응답 구조에서 items 키를 찾지 못하는 KeyError가 발생했습니다.

  • 원인: API 키 오설정 또는 호출 한도 초과 시 에러 메시지가 반환됨
  • 해결: HTTP 상태 코드를 선제적으로 확인하도록 로직 보강 
response = requests.get(url, headers=headers)
if response.status_code != 200:
    print(f"API 오류: {response.status_code}")

03. 개발 환경 구축 이슈

3.1. NumPy 버전 충돌 (v1.x vs v2.x)

NumPy 2.x 업그레이드 이후 기존 라이브러리와의 호환성 문제가 발생했습니다. 

# 해결 방법: 가상환경 재생성 및 버전 고정
pip uninstall numpy -y
pip install "numpy<2"

3.2. Streamlit 도입 중단

Python 버전 불일치로 인한 설치 실패 및 UI 확장성 부족 판단 하에 Flutter 집중으로 선회하였습니다.

04. Flutter 앱 구현 이슈

4.1. 비동기 콜백 함수 처리

화살표 함수 내에서 await를 직접 사용하여 발생한 구문 오류를 수정했습니다. 

// 해결: 콜백을 async 블록으로 래핑
onZoomIn: () async {
  final zoom = await _currentZoom();
  _mapController?.updateCamera(NCameraUpdate.withParams(zoom: zoom + 1));
}

4.2. ADB 장치 승인 미처리

물리 기기 디버깅 시 device unauthorized 에러가 발생했습니다. 개발자 옵션에서 USB 디버깅 승인을 초기화하고 재연결하여 해결했습니다.

03. 모델 선정

3.1 초기 모델 (KoBERT)

  • 모델: skt/kobert-base-v1
  • 입력: review_description

문제점

  • AUC: 0.785 (낮음)
  • y_prob 범위: 0.993 ~ 0.997 (분포 붕괴)
  • 토크나이저 충돌 발생

3.2 개선 모델 (KcBERT)

  • 모델: beomi/kcbert-base
  • 입력: title + [SEP] + description

개선 효과

  • AUC: 0.949
  • y_prob 범위: 0.002 ~ 0.994
  • HuggingFace 호환성 확보

3.3 성능 비교

항목 KoBERT KcBERT
AUC 0.785 0.949
입력 description title + description
확률 범위 좁음 넓음

05. OpenVINO 변환

5.1 변환 구조

PyTorch (.pt)
├─ ONNX (FP32)
│   ├─ OpenVINO FP16
│   └─ OpenVINO INT8 (권장)

⚠️ FP16 → INT8 금지

5.2 PyTorch → ONNX

torch.onnx.export(
   model,
   (
      dummy["input_ids"],
      dummy["attention_mask"],
      dummy["token_type_ids"]
   ),
   "kcbert_final.onnx",
   opset_version=14,
   do_constant_folding=True
)

5.3 ONNX → FP16

ovc kcbert_final.onnx \
   --output_model ./kcbert_openvino/kcbert_final \
   --compress_to_fp16 True

5.4 ONNX → INT8

quantized_model = nncf.quantize(
   model,
   nncf.Dataset(list(get_calibration_dataset()))
)

07. 임계값 및 등급 분류

7.1 기준 (FP16)

  • Upper: 0.0074
  • Lower: 0.5979

7.2 등급 정의

등급 조건 의미
상 ≤ 0.0074 비광고 확실
중 ≤ 0.5979 애매
하 > 0.5979 광고 의심

7.3 코드

def classify_grade(prob, upper=0.0074, lower=0.5979):
   if prob <= upper:
       return "상"
   elif prob <= lower:
       return "중"
   else:
       return "하"

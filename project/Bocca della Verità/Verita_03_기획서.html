진실의 입 (The Truth-Filtering Engine)

AI 필터링 기술 기반 맛집 광고 리뷰 식별 서비스

무분별한 광고성 리뷰 속에서 인공지능을 통해 '진짜 후기'만을 선별하고, 정보 신뢰도를 회복하여 사용자의 효율적인 의사결정을 돕는 미식 가이드를 제안합니다.

01 — Overview

프로젝트 개요

네이버 지도 리뷰의 광고 확률을 AI로 분석하여 사용자에게 투명한 정보를 제공하고, 광고가 제거된 진짜 리뷰만을 빠르게 요약해 주는 서비스입니다.

What문제 인식
마케팅 업체의 바이럴 공세로 인한 '대가성 후기' 범람 및 신뢰도 상실
Why발생 원인
정교화된 바이럴 기법(이미지 배너 숨기기), 정보 불균형 및 필터링 기능 부재
02 — Differentiation

서비스 특화 포인트

구분 기존 서비스 (네이버 지도 등) 본 프로젝트 (진실의 입)
정보 정렬 최신순, 관련도순 중심 신뢰도 및 광고 비율 중심
검증 방식 사용자가 수동으로 확인 AI가 진실 여부 % 고지
검색 시간 평균 15~20분 소요 5분 이내 요약 완료
03 — Goals

프로젝트 목표

01
정확도
광고 탐지 정확도 70% 이상 달성 (테스트 셋 기준)
02
속도
링크 입력 후 결과 출력까지 3분 이내 완료
03
가치
유의미한 가비지 데이터 제거를 통한 의사결정 피로도 해소
04 — Core Features

핵심 기능 정의 (MVP)

리뷰 데이터 추출 Must
네이버 지도 링크 입력 시 관련 포스팅 텍스트 및 메타데이터 자동 크롤링
광고 농도 분석 Must
AI 문맥 분석을 통해 해당 식당의 전체 리뷰 중 광고 비중을 수치화
클린 리뷰 요약 Must
광고를 제외한 '찐 후기'들만 모아 핵심 장/단점을 AI로 자동 추출 및 요약
05 — Technology

기술 스택 활용 계획

  • AI/ML: GPT API / BERT — 문맥 분석 및 광고성 말투 탐색
  • Scraping: Selenium / BeautifulSoup — 데이터 수집
  • Backend: Python / FastAPI — 분석 알고리즘 처리
  • Frontend: React — 분석 대시보드 및 리포트 시각화

Summary

종합 요약

본 프로젝트는 정보 과부하와 무분별한 광고성 리뷰로 발생하는 사용자 피로도를 해결하기 위한 "신뢰 필터링 엔진"입니다. 소비자가 광고에 속지 않는 건전한 외식 환경을 조성하고, 데이터 기반의 객관적 지표를 통해 신뢰도 높은 미식 생활을 지원하는 것을 최종 목표로 합니다.

01 — Overview

프로젝트 개요

배경 및 목적

국내 식음료 업장(카페, 한식·중식·이탈리안·일식 레스토랑 등)은 감성적·외국어 표현으로 이루어진 메뉴명을 폭넓게 사용하고 있어, 시니어·발달장애인·외국인 등 정보 소외계층이 메뉴를 이해하는 데 어려움을 겪고 있습니다.

'그게 뭔데(What The Menu)'는 스마트폰 카메라로 메뉴판을 비추면 Vision AI와 LLM이 텍스트를 인식하여 직관적인 우리말(또는 선택 언어)로 실시간 해독·안내하는 서비스입니다. 카페뿐 아니라 중식당의 한자 메뉴, 이탈리안 레스토랑의 이탈리아어 메뉴, 일식당의 히라가나·가타카나 메뉴 등 다양한 업종·언어를 포괄하며, 정보 접근 불평등을 해소하여 UN SDG Goal 10(불평등 감소) 달성에 기여합니다.

주요 사용 시나리오

Scenario 01 비회원 시나리오
  • 사용자가 앱을 실행하고 카메라를 메뉴판에 향한다.
  • Vision AI가 메뉴 텍스트(한국어, 한자, 가나, 로마자 등)를 실시간으로 인식한다.
  • LLM이 업종을 자동 감지하고 인식된 텍스트를 직관적인 설명으로 변환한다.
  • 해독된 텍스트가 AR 오버레이로 화면에 표시된다.
  • 하루 3회 해독 후 추가 사용 시 회원가입 안내 팝업이 표시된다.
Scenario 02 회원 시나리오
  • 로그인 후 제한 없이 해독 가능.
  • 특정 메뉴를 탭하면 예시 사진과 알레르기 정보가 표시된다.
  • 해독 이력이 자동 저장되어 My 이력에서 확인 가능 (업장명·날짜 포함).
  • My 알레르기에 사전 설정된 항목이 메뉴 해독 시 강조 표시된다.

06 — Rate Limiting & Cost Policy

API 비용·사용량 제한 정책

6.1 사용량 제한 (Rate Limiting) 정책

구분 일일 해독 횟수 프롬프트 토큰 상한 (1회) 이미지 검색 API 비고
비회원 3회 500 tokens 3회 (해독과 동일 쿼터) 기기 ID 기반 카운트
회원 (무료) 무제한* 1,000 tokens 무제한* 계정 기반 카운트
* 어뷰징 방지 분당 10회 초과 시 1분 차단 — — 비정상 패턴 감지 시 계정 검토

6.2 비용 관리 정책

  • Vision AI 호출: 이전 프레임과 유사도 80% 이상이면 캐시 결과 재사용 (서버 사이드 캐싱)
  • LLM 호출: 동일 업종 + 동일 메뉴명 조합의 해독 결과는 24시간 캐시하여 중복 호출 비용 절감
  • 이미지 API: CDN 캐시 적용, 동일 메뉴명 이미지는 7일간 재사용
  • 업종별 캐시 분리: 같은 메뉴명이라도 업종이 다르면 별도 캐시 (예: 카페의 'Americano' vs 이탈리안의 'Americano')
  • 월별 API 비용 대시보드를 내부 운영하며, 예산 초과 시 비회원 쿼터를 일시적으로 축소 가능

6.3 어뷰징 방지

  • 비회원: 기기 ID(Device Fingerprint) + IP 주소 조합으로 하루 3회 쿼터 관리
  • VPN/프록시 우회 감지 시 쿼터를 기기 ID 단독 기준으로 전환
  • 자동화 봇 탐지: 요청 간격이 0.5초 미만인 경우 CAPTCHA 인증 요구

09 — Constraints & Assumptions

제약 조건 및 가정

9.1 기술 제약

  • Vision AI 및 LLM 호출은 네트워크 연결 필요 (완전 오프라인 미지원)
  • OCR 인식률은 메뉴판 폰트·조명·언어 환경에 따라 변동 가능 (한자·가나는 영문 대비 5~10% 낮을 수 있음)
  • LLM 해독 결과는 참고 정보이며 100% 정확성 보장 불가
  • AR 오버레이는 ARKit(iOS) / ARCore(Android) 지원 기기에서만 동작
  • 업종 자동 감지는 메뉴판 텍스트가 충분하지 않을 경우 (항목 3개 미만) 정확도가 낮아질 수 있음
  • 중국어 번체 입력은 간체 전환 후 LLM에 전달 (번체 직접 처리는 Phase 2)

9.2 비즈니스 제약

  • 알레르기 정보는 '가능성 안내' 수준으로만 제공하며 의료적 근거로 사용 불가
  • 이미지 검색 API의 저작권 정책 준수 필요
  • Phase 1: 카페 + 주요 레스토랑 업종 (한식·중식·일식·이탈리안·프렌치) 대상
  • 편의점·마트·패스트푸드 체인 등 비레스토랑 업종은 Phase 3 이후 검토
  • 비회원 쿼터(하루 3회)는 API 비용 구조 변경 시 조정 가능

9.3 가정 사항

  • 사용자는 스마트폰과 카메라 앱의 기본 조작 능력을 보유함
  • 앱 사용 중 최소 3G 이상의 네트워크 환경 제공됨
  • 메뉴판은 본 서비스가 지원하는 6개 언어 중 하나 이상으로 작성됨

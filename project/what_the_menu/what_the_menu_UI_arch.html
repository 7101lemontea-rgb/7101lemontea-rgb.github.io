2.1 디자인 시스템 기본값

항목 기본값 접근성 기준
기본 폰트 크기 16sp (본문) / 20sp (제목) 최소 12sp. 사용자 설정 4단계 지원
색상 대비비 전경:배경 최소 4.5:1 (AA 기준) WCAG 2.1 AA 준수

2.2 하단 탭바 (Global Navigation)

📷 스캔 탭

기본 진입 탭. SCR-003 이동

📋 이력 탭

회원은 SCR-008, 비회원은 SCR-007(로그인 유도). 탭바에 🔒 아이콘 오버레이.

⚙️ 설정 탭

SCR-009 이동. 선택된 탭: 아이콘 Primary Blue + 텍스트 Bold. 미선택: Gray + Regular.

2.3 공통 상단 헤더

화면 유형 좌측 중앙 우측
루트 화면 (스캔·이력·설정) 없음 (탭바로 이동) 앱 로고 또는 화면명 없음

03 — Screen Specifications

화면별 상세 설계

SCR-001 스플래시 / 앱 초기화

역할
앱 초기화 (로컬 캐시 로드, 인증 상태 확인, 리소스 프리로드)
노출 시간
최소 1.0초 ~ 최대 2.5초 (초기화 완료 시 자동 전환)
전환 대상
최초 설치 → SCR-002 / 재방문(비회원) → SCR-003 / 재방문(회원, 토큰 유효) → SCR-003 / 토큰 만료 → SCR-007
접근성
화면 리더: 'What The Menu 앱 로딩 중' 발화
애니메이션
로고 0.3초 fade-in, 슬로건 0.2초 지연 후 fade-in

SCR-002 온보딩 (최초 1회)

역할
서비스 핵심 가치 설명 (3~4 페이지 카드 슬라이드)
페이지 구성
① 카메라로 메뉴판 해독 ② 업종별 맞춤 설명 ③ 알레르기 안내 ④ 회원가입 유도
제스처
좌우 스와이프 또는 '다음' 버튼
건너뛰기
우측 상단 ✕ 또는 하단 '건너뛰기' → SCR-003 직행
저장
완료 시 로컬에 '온보딩 완료' 플래그 저장 (재노출 방지)
접근성
페이지 인디케이터에 'n페이지 중 m페이지' aria-label

SCR-003 메인 스캔 화면 (AR 해독) ⭐ 핵심 화면

핵심 화면 (FR-001 / FR-002 / FR-003) — 카메라 전체화면 + AR 오버레이가 앱의 메인 가치를 전달하는 화면입니다. 사용자가 가장 오래 머무르며, 접근성과 성능이 가장 중요하게 적용됩니다.

카메라 프리뷰
전체 화면 라이브 스트림. Safe Area 침범 없음.
업종 뱃지
상단 중앙. 감지된 업종 아이콘+이름. 탭하면 업종 수동 변경 팝업
AR 오버레이
원문 위치에 해독 박스 렌더링. 박스 배경 #000 60%. 텍스트 #FFF
🔔 알레르기 아이콘
해독 박스 우측. My 알레르기 해당 항목 있으면 빨간 🔔 표시
하단 퀵 액션바
TTS(음성재생) / 언어 전환 / 알레르기 전체 보기 버튼
하단 탭바
전체화면이지만 탭바는 상시 노출 (반투명 배경)
설정 단축
우상단 ⚙️: 접근성·언어 설정 빠른 진입

SCR-003 상태 분기

상태 화면 표현 UX 처리
초기 (카메라 권한 없음) 권한 요청 오버레이 표시 '카메라 권한이 필요합니다' + 설정으로 이동 버튼
스캔 중 (OCR 처리 중) 해독 박스에 점 로딩 애니메이션 (···) 1초 이내 완료 목표. 초과 시 스피너 노출
해독 완료 AR 오버레이 0.2초 fade-in 표시 TTS 설정 시 자동 읽기
캐시 결과 표시 해독 박스 좌측 상단에 🕐 아이콘 '저장된 결과' 툴팁
비회원 2회 사용 하단에 '오늘 무료 1회 남음' 토스트 배너 회원가입 유도 (소프트)
비회원 3회 초과 다음 해독 시도 시 SCR-006 팝업 트리거 스캔 일시 차단
네트워크 없음 상단에 '오프라인' 배너. 캐시 결과만 표시 '인터넷 연결 시 실시간 해독 가능' 메시지
저조도 환경 '조명이 어둡습니다' 토스트 + 화면 밝기 자동 최대화 제안 Flash 켜기 버튼 노출

SCR-003 접근성

  • 화면 리더: AR 오버레이 텍스트를 포커스 순서대로 발화 (위→아래, 좌→우)
  • TTS 버튼: 현재 화면 해독 결과 전체를 순서대로 읽기. 재생 중 다시 탭하면 중지
  • 고대비 모드: AR 박스 배경 #000 80% / 텍스트 #FFD700 (대비비 8:1↑)
  • 글자 크기 '매우큰': AR 해독 텍스트 22sp → 28sp, 박스 크기 자동 확장

SCR-004 메뉴 상세 팝업 (FR-004 / FR-005)

진입 방법
SCR-003 AR 오버레이 박스 탭 → Bottom Sheet (슬라이드업)
이미지 캐러셀
최대 3장. 스와이프 전환. 이미지 로딩 중 shimmer 효과. 실패 시 플레이스홀더
메뉴 정보
원문(원어) + 해독명 + 업종/지역 설명 1줄
재료 섹션
LLM이 추출한 주요 재료 목록
알레르기 섹션
해당 항목 아이콘 그리드. My 알레르기 해당 항목은 빨간 테두리 강조
면책 문구
'실제 성분은 업장마다 다를 수 있습니다' 상시 노출 (FR-005)
신고 버튼
하단 '이 설명이 잘못됐어요' → SCR-013 진입
닫기
✕ 버튼 또는 배경 탭 또는 아래로 스와이프

SCR-005 알레르기 경고 팝업

진입
SCR-004 알레르기 섹션 탭 또는 SCR-003 🔔 아이콘 탭
내용
My 알레르기에 등록된 항목 중 해당 메뉴에 포함 가능한 항목 강조
면책 문구
의료적 판단 불가 명시. '직원에게 직접 확인' 안내 (FR-005)
CTA
'확인했습니다' 버튼 → 팝업 닫기
My 알레르기 미설정 시
'My 알레르기를 설정하면 맞춤 경고를 받을 수 있어요' + 설정 링크
접근성
경고 팝업 진입 시 화면 리더 자동 포커스. 진동 햅틱 피드백

SCR-006 비회원 쿼터 초과 팝업 (FR-007)

트리거
비회원 해독 3회 완료 후 다음 해독 시도 시 자동 표시
메시지 톤
강압적이지 않은 긍정적 유도. '오늘 다 사용' 프레임
혜택 목록
무제한 해독 / 이력 30건 저장 / My 알레르기 — 3가지만 명시
주 CTA
'무료로 시작하기' → SCR-007 (로그인)
보조 CTA
'내일 다시 오기' → 팝업 닫기. 스캔은 내일까지 불가 (토스트 안내)
카운트 리셋
자정 자동 리셋. 리셋 시 앱 상태 업데이트

SCR-007 로그인 / 회원가입 (FR-007)

진입 경로
SCR-006 CTA / 이력 탭 탭 / 설정에서 '로그인'
로그인 방식
Apple ID / Google / Kakao (OAuth 2.0 버튼 순서: iOS는 Apple 최상단)
신규/재방문
동일 화면. 신규는 자동 계정 생성, 재방문은 기존 계정 연결
로그인 성공
직전 화면(진입 경로)으로 복귀. 토스트: '로그인 완료'
닫기 방법
SCR-006 경유: ✕ (팝업 닫기) / 직접 진입: ← (뒤로가기)
법적 동의
소셜 로그인 버튼 하단에 약관 동의 문구 상시 노출
접근성
각 소셜 버튼 aria-label: '애플 계정으로 로그인' 등

SCR-008 해독 이력 (회원 전용) (FR-007 / FR-008)

목록 구성
날짜 그룹핑. 최신순 정렬. 업종 아이콘 + 원문 + 해독명
최대 건수
최근 30건. 초과 시 가장 오래된 항목 자동 삭제
알레르기 표시
My 알레르기 해당 항목 있는 이력에 🔔 뱃지
검색
원문 또는 해독명으로 전문 검색
탭 동작
이력 항목 탭 → 원문+해독+재료+알레르기 상세 팝업 (SCR-004 재사용)
스와이프 삭제
좌로 스와이프 → 삭제 확인 (iOS: swipe action / Android: contextual menu)
비회원 접근 시
이 탭 탭하면 SCR-007(로그인) 이동. 탭바에 🔒 아이콘
빈 상태
'아직 해독 이력이 없어요. 스캔 탭에서 시작하세요!' + 스캔 CTA

SCR-009 내 설정

계정 섹션
로그인 시 이메일/닉네임 표시 + 로그아웃. 비로그인 시 '로그인하기' 버튼
출력 언어
현재 설정값 인라인 표시. 탭 → SCR-011(언어 설정)
업종 즐겨찾기
등록된 업종 수 인라인 표시. 탭 → 업종 선택 시트
My 알레르기

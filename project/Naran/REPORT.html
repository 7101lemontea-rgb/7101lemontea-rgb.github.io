01 — Overview

프로젝트 개요

주제: 이중언어(한국어·베트남어) 기반 AI 튜터를 활용하여 다문화 가정 부모와 자녀가 함께 교과 개념을 학습할 수 있는 동반 학습 지원 서비스

기획 의도

  • 문제 인식: 부모의 한국어 수준 저하로 인한 자녀 학습 지도 참여 불가능
  • 원인 분석: 기존 학습 도구의 다국어 지원 부재 및 학생 단독 사용 중심 구조
  • 해결 방향: 이중언어 학습 서비스 및 부모용 학습 리포트 제공을 통한 참여 유도

차별성 (Distinction)

구분 기존 서비스 에듀 프렌드
핵심 기능 문제 정답 제공 (한국어 전용) 이중언어 개념 해설
부모 참여 없음 부모용 이중언어 가이드
학습 방식 학생 단독 학습 부모·자녀 동반 학습
02 — Project Goals

프로젝트 목표

정량적 목표 KPI
  • 핵심 기능 구현율 100% (시연 기준)
  • AI 응답 시간 5초 이내 달성
  • 이중언어(한-베) 설명 적절성 팀 내 검수 통과
기대 효과 Expected Effect

언어 장벽 없는 교과 개념 이해 및 자기주도 학습 역량 향상. RAG 기반 AI 개발 역량 확보.

03 — Core Features (MVP)

핵심 기능 정의

01
이중언어 AI 질의응답 RAG + GPT
학생(한국어)과 부모(베트남어)의 질문에 맞춤형 설명과 예시 제공 (Must Have)
02
문제 풀이 화면 UI Interaction
교과서 기반 문제 제공 및 즉시 질문 가능한 플로팅 채팅 오버레이
03
오답 개념 재설명 Feedback
오답 시 연관 개념 자동 재설명 및 부모용 지도 가이드 제공

Summary

종합 요약

본 프로젝트는 다문화 가정의 언어 장벽에 의한 학습 소통 단절을 해결하기 위해, 학생과 부모가 각자의 언어로 교과 개념을 즉시 이해하고 함께 공부할 수 있는 대화형 학습 도우미 개발을 목표로 합니다. RAG 기술을 활용한 신뢰도 높은 개념 검색을 통해 교육 격차 해소라는 사회적 가치를 실현하겠습니다.

PlayerMovement.cs

Assets/Scripts/PlayerMovement.cs 생성

using UnityEngine;
using UnityEngine.EventSystems;

public class PlayerMovement : MonoBehaviour
{
    public float moveSpeed = 3f;

    void Update()
    {
        // InputField에 포커스가 있으면 이동 불가
        if (EventSystem.current != null && EventSystem.current.currentSelectedGameObject != null)
            return;

        float h = Input.GetAxisRaw("Horizontal");
        float v = Input.GetAxisRaw("Vertical");
        Vector3 move = new Vector3(h, v, 0).normalized * moveSpeed * Time.deltaTime;
        transform.Translate(move);
    }
}

트러블슈팅: Input System 오류

오류
InvalidOperationException: You are trying to read Input using the UnityEngine.Input class, but you have switched active Input handling to Input System package
원인
프로젝트가 새 Input System만 사용하도록 설정되어 있었음
해결
Edit → Project Settings → Player → Other Settings → Active Input Handling → Both
팀 공유
이 설정은 ProjectSettings/ProjectSettings.asset에 저장되어 Git으로 공유되므로 팀원들은 별도 설정 불필요

Hiki 선택 → Add Component → Player Movement. Rigidbody 2D는 불필요 (Transform.Translate 방식 사용).

CameraFollow.cs

Assets/Scripts/CameraFollow.cs 생성

using UnityEngine;

public class CameraFollow : MonoBehaviour
{
    public Transform target;
    public float smoothSpeed = 5f;

    void LateUpdate()
    {
        if (target == null) return;
        Vector3 targetPos = new Vector3(target.position.x, target.position.y, transform.position.z);
        transform.position = Vector3.Lerp(transform.position, targetPos, smoothSpeed * Time.deltaTime);
    }
}

Main Camera → Add Component → Camera Follow. Target 항목에 Hiki 드래그.

트러블슈팅: 한국어 폰트 문제

증상
The character with Unicode value \uXXXX was not found in the [LiberationSans SDF] font asset
한국어 텍스트가 □로 표시됨

해결: 맑은 고딕 폰트 에셋 생성

① 폰트 Import
C:\Windows\Fonts\malgun.ttf → Unity Assets 폴더로 드래그
② Font Asset Creator Window → TextMeshPro
  • Source Font: MALGUN
  • Atlas Resolution: 4096 × 4096 (512는 한글 전체를 담기 부족)
  • Character Set: Custom Range
  • Character Sequence: 32-255,12593-12686,44032-55203
③ 적용
Generate Font Atlas → Save 후, DialogueText · InputField의 모든 Text 컴포넌트에 MALGUN SDF 적용
Character Sequence 범위 설명
32-255 기본 라틴 영문자, 숫자, 기본 기호
12593-12686 한글 자모 ㄱ, ㄴ … ㅏ, ㅑ
44032-55203 한글 음절 가, 나, 다 …

주의 — Character Sequence는 10진수(Decimal) 형식. 16진수(0020-00FF) 입력 시 FormatException 오류 발생.
